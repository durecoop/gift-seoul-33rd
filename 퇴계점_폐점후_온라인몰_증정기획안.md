| 항목 | 내용 |
|------|------|
| **폐점 매장** | 퇴계점 |
| **폐점일** | 2026년 3월 25일(수) |

- 퇴계점 폐점으로 오프라인 이용이 어려워진 조합원님들의 **구매 이탈 방지**
- 온라인쇼핑몰 **첫 이용 경험** 제공을 통한 자연스러운 전환
- 주 1회 증정을 통해 **정기적 온라인 주문 습관** 형성
- 서울남부와 동일 품목 증정으로 **매장 간 혜택 형평성** 유지

## 3. 운영 방안

### 3-1. 증정 조건

| 항목 | 내용 |
|------|------|
| **대상** | 퇴계점 이용 조합원 (온라인쇼핑몰 주문 시) |
| **증정 주기** | 주 1회 |
| **증정 방식** | 5품목 일괄 증정 (1인 1회/주) |
| **최소 주문금액** | 100,000원 이상 (배송일 기준) |
| **증정 기간** | 2026년 3월 31일(월) ~ 4월 25일(금) [4주간] |
| **적용 기준** | 배송일 기준 (주문일 아닌 배송일 기준) |

### 3-2. 최소 주문금액 설정 근거

서울남부 요일별 증정 최소 주문금액:
- 월(느타리버섯): 3만원 / 화(당근): 3만원 / 수(쑥버무리): 5만원 / 목(유정란): 7만원 / 금(삼치): 10만원

→ 5품목 묶음 증정이므로 **최고 기준금액인 10만원**을 최소 주문금액으로 설정
→ 10만원 이상 주문 시 5품목 전체를 한 번에 증정하여 **체감 혜택 극대화**

## 7. 유의사항 및 운영 규칙

1. **1인 1회/주 제한** — 동일 조합원 주당 1회만 증정 (중복 수령 불가)
2. **배송일 기준 적용** — 주문일이 아닌 배송일 기준으로 해당 주차 판단
3. **퇴계점 조합원 식별** — 조합원 DB 기반으로 퇴계점 등록 조합원 대상 자동 적용
4. **품목 소진 시** — 해당 주차 증정 조기 종료 (선착순 안내)
5. **기간 연장 검토** — 4주 운영 후 전환율 및 재주문율 분석하여 연장 여부 결정
